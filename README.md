# SIMPLE GULP CONFIG

## Stack

```
* GULP
* SCSS - CSS preprocessor
* NUNJUCKS - Template engine
* SCSS Lint
* CSS minify
* BrowserSync - Reload when code change
* Allow ES6 syntax
```

## Installation

### Install dependencies
```
cd /path/to/folder && npm install || yarn install
```

## Usage
### Start server
```
gulp watch
```

### Watch files change
```
gulp watch
```

### Build files
```
gulp build
```

### Deploy
```
gulp deploy
```
